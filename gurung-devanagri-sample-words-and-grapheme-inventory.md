## 1\. How to use (type/write) Nepali Romanized pro Keyword {#1.-how-to-use-(type/write)-nepali-romanized-pro-keyword}

On macOS: 

- change input method to Nepali (Romanized) Pro   
- if needed, enable “Show Keyboard Viewer”

Key sequences: 

- Two keys joined by ‘+’: hold down first key while press second key, then release together   
- Keys joined by space: press keys in sequence
